## 部署（GitHub Pages）

1. 將此 repo 推送至 GitHub
2. 進入 **Settings → Pages**
3. Source 選 **Deploy from a branch**，Branch 選 `main`，資料夾選 `/ (root)`
4. 儲存後約 1–2 分鐘即可在 `https://<your-username>.github.io/<repo-name>/` 存取
